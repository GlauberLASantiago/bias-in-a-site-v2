Guia de Uso do App
  • 1. Montar Cifras: Selecione a fundamental no piano virtual ou no braço do violão (toque no mesmo botão de alteração para alternar sustenido/bemol) e a qualidade (maj7, 9, etc.) para criar o acorde no slot ativo.
  • 2. Substituir & Foco: Clique em um acorde na sequência (borda roxa) para focar. O piano e o violão mudarão para a fundamental dele. Selecione outra qualidade para trocar. Aperte Esc para tirar o foco.
  • 3. Visor de Notas: O visor na parte inferior exibe as notas do acorde em foco com enarmonia teórica exata (ex: mi# no C#9). Passe o mouse e clique nas notas para ouvi-las individualmente em escala ascendente.
  • 4. Controles de Play: Use Tocar do Início para iniciar a música do primeiro compasso ou Tocar do Selecionado para reproduzir a partir do acorde em foco atual.
  • 5. Dividir / Unir: Dê duplo clique em um slot para dividir o compasso (2 tempos) ou unir novamente (4 tempos).
  • 6. Teclado & Desfazer/Refazer: Digite cifras no slot selecionado (ex: digite C, depois m7 e Enter). Use Ctrl+Z (Desfazer) e Ctrl+Y (Refazer) para editar.
  • 7. Copiar, Colar & Apagar: Selecione compassos e use Ctrl+C / Ctrl+V. Passe o mouse sobre o slot e clique no x vermelho no canto para apagá-lo.
  • 8. Cadência Automática: Ao inserir um acorde dominante (`9` ou `7b9`), se o próximo slot de compasso estiver vazio, o sistema pré-selecionará automaticamente no piano e no violão a nota fundamental correspondente a uma quarta justa acima (por exemplo, de G para C), facilitando a montagem rápida de cadências harmônicas.

Explicação dos Formatos de Exportação

  • • Projeto (.bias): Salva localmente sua sequência harmônica em formato JSON simplificado (apenas cifras e tempos). Use o botão "Abrir Projeto" para carregar este arquivo e continuar a editar de onde parou.
  • • Moodle (HTML): Gera o código HTML pronto para tarefas ou fóruns do Moodle com uma grade de partitura elegante contendo cifras e compassos numerados. Possui as opções **MP3 Embutido (Base64)** para áudio integrado no próprio código e **Online (Sem Base64)** para um player dinâmico extremamente leve que carrega os áudios diretamente da web.
  • • Áudio (.mp3): Exporta e baixa o áudio completo consolidado do acompanhamento da sequência em formato MP3 estéreo comprimido de alta qualidade. *Disponível após tocar a música pelo menos uma vez.*
  • • Imagem (.png): Baixa uma folha de partitura visual com cifras centralizadas, numeração de compassos (com frações como `1.3` nos splits), título, autor, data e horário exato de geração do arquivo no canto.
  • • Tabela (MD): Fornece o código de uma tabela em formato Markdown para colar em relatórios, arquivos readmes, ou documentos compatíveis com formatação de tabela simples.
  • • MusicXML: Exporta em formato de notação musical `.xml` padrão da indústria, ideal para abrir e diagramar a grade harmônica em programas como MuseScore, Sibelius ou Finale.
  • • BIAB: Exporta uma string de texto limpa contendo as cifras e tempos no formato legível de importação rápida do software Band-in-a-Box.
